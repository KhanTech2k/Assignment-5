Important FAQ

What is the Document Object Model(DOM)?

The DOM is a tree-like structure representing a webpage's elements, enabling dynamic manipulation via programming languages like JavaScript.

How do you select an element from the DOM?

You can select a DOM element using methods like `getElementById()`, `querySelector()`, `getElementsByClassName()`, or `getElementsByTagName()`.

What is event delegation in the context of the DOM ,and why is it useful?

Event delegation allows handling events at a parent level, improving efficiency by avoiding multiple event listeners for child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes with `setAttribute()` and modify styles directly using `element.style.property` in the DOM.
